## What this session covers

Learn the foundational concepts of Rust programming including variables, mutability, data types, functions, conditionals, and loops. This lesson introduces how Rust handles data safely and why immutability is a core part of the language.

## Topics

- Variables and `mut`
- Loops (`loop`, `while`, `for`)
- Functions
- Data types (`i32`, `u64`, `bool`, `String`, `&str`)

## Goal

Write and run basic Rust programs using variables, loops, and functions.
